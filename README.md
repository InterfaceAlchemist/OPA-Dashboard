# OPA Dashboard

A geospatial agricultural dashboard for the **Office of Provincial Agriculture (OPA)** of Pampanga, built as a Proof of Concept (POC) frontend.

## Features

- 📊 Interactive statistics cards (Farmers, Farms, Agriculture, Fishery, Livestock, Organizations)
- 🗺️ Full-width GIS map hero with overlaid stats using MapLibre GL
- 🛰️ Himawari-8 live weather satellite animation overlay
- 📈 Charts and data visualizations using Chart.js
- 🌙 Dark/Light mode toggle
- 🎨 3 Color themes (Default, Agriculture, Ocean)
- 📋 Damage Assessment Summary table
- 👥 Recent Registrations & Interventions
- ⚙️ Theme settings panel with font size controls

## Built With

- PHP
- Bootstrap 5
- MapLibre GL JS 4.7.1
- Turf.js 7.1.0
- Chart.js
- Font Awesome
- Bootstrap Icons

## Setup

1. Clone the repository
2. Place the project in your WAMP/XAMPP `www` or `htdocs` folder
3. Open `localhost/itbs-proj-opa` in your browser
